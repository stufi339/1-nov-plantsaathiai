üîç Test 2: Response Time

Real Satellite Data:

5-15 seconds
Real Google Earth Engine queries take time to:
  • Authenticate with GEE servers
  • Query Sentinel-2 image collection
  • Apply cloud masking
  • Calculate vegetation indices
  • Reduce to mean values

Simulated Data:

Instant (< 1 second)
Simulated data is calculated locally using algorithms, so it returns almost instantly.

üîç Test 3: Data Consistency

Real Satellite Data:

CONSISTENT
Test: Refresh the same field multiple times within 1 hour.

Expected: Values should be EXACTLY the same (satellite images don't change that fast).

Real satellite data is cached and only updates when new imagery is available (every few days).

Simulated Data:

VARIES SLIGHTLY
Test: Refresh the same field multiple times.

Expected: Values change slightly each time (¬±5-10%).

Example: NDVI might be 0.72, then 0.74, then 0.71 on consecutive refreshes. This happens because Math.random() adds variation to simulate real-world variability.
